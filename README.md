# Recipe


主要功能：


－瀏覽食譜（包含成分、步驟、照片分享）

－關鍵字搜尋食譜

－語音辨識切換步驟：使用OpenEars，使用者可自行設立內容(keywords)及觸發的事件(上一個步驟或下一個步驟)，透過語音辨識內容進行步驟的切換


   -------------------以下為登入的使用者可使用的功能-------------------

－fb登入：使用FBSDK，使用者可連接fb登入app

－拍照上傳分享圖片：料理完成之後可以拍照分享至該則食譜的照片分享區

－新增食譜（上傳相片、新增成份、新增步驟）

－編輯已發布食譜

－刪除已發布食譜

－收藏食譜
